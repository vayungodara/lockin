## Notes

**URGENT: Next.js 16.1.2 has 21 known security vulnerabilities** (9 high, 9 moderate, 3 low) with fixes available in 16.2.6. This is the most critical finding. Key issues include:
- **SSRF via WebSocket upgrades** (CVSS 8.6) — attacker can reach internal services
- **Middleware/Proxy bypass** (CVSS 8.1, 7.5) — multiple vectors to bypass auth middleware
- **Multiple DoS vectors** (CVSS 7.5) — Server Components, Cache Components, Image API

**Recommended action:** Manually update `next` from 16.1.2 to 16.2.6 along with `eslint-config-next` to 16.2.6 as soon as possible. Test thoroughly — this is a minor version bump with security patches.
