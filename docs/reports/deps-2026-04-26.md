Since `next` is a protected/pinned package, this must be done manually. **Strongly recommend updating next to 16.2.4 and eslint-config-next to 16.2.4 as soon as possible.** Key vulnerabilities include:

`@vercel/analytics` and `@vercel/speed-insights` have major version updates (v2). Review their changelogs before upgrading — these may have breaking API changes.
